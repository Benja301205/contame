`Quote` renders a verbatim respondent quote with contame's hot top border and italic-serif body. Use in a 2-up grid for "citas destacadas".

```jsx
<Quote category="Lo que más gustó" meta="Verbatim · sin corrección">
  La energía del cierre fue otra cosa — me llevo eso.
</Quote>
```

Set `mark={false}` to drop the decorative quotation glyph in tight layouts.
