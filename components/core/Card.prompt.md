`Card` is the brand's hairline surface — hard corners, 1px border, optional hot top accent.

```jsx
<Card accent fill="gradient" padding={32}>
  <Eyebrow>Pulso</Eyebrow>
  <h3>Lardito</h3>
</Card>
```

Use `accent` for the 2px hot top bar (seen on team/pain cards), `fill="fill"` for a solid `--bg-2` surface, `fill="gradient"` for the lit-corner gradient. Default is transparent over the page.
