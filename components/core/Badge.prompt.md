`Badge` shows a status pill or a solid severity tag.

```jsx
<Badge status="online">En línea</Badge>
<Badge status="live">En vivo</Badge>
<Badge variant="tag" tone="alta">Alta</Badge>
<Badge variant="tag" tone="soft">Nuevo</Badge>
```

Pills carry a colored dot; tags are solid fills. Use `tone="alta|media|baja"` for severity, `soft` for a low-emphasis chip.
